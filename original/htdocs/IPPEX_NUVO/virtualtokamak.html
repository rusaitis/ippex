Virtual Tokamak

Welcome to the Virtual Tokamak section. A tokamak is a machine that confines and heats plasma in order to produce fusion energy. These modules put you in the control seat of a tokamak fusion reactor. You will explore some of the variables relevant to fusion research and tune a tokamak to make it run optimally.

Magnetic Drifts

This applet shows how magnetic fields control and contain a plasma when it is too hot to contain by any other conventional methods. To get the longest containment time, you can manually and automatically turn on/off magnetic controls.

Time Dependent Tokamak

This applet lets you plug in waveforms for all three parameters. The output is likewise shown as a graph so you can see how the plasma changes over time.

Steady State Tokamak

This version of the Virtual Tokamak allows you to type in the parameter values as well as use the slider bars. More control of the reactor means better performance which means an even higher score!
