Operate Your Own Tokamak Reactor!

*NEW* 2005 Version!

As part of the Internet Plasma Physics Education eXperience (IPPEX) project, this Java applet is designed to illustrate the basic principles of magnetically confined fusion.

Anxious to get started? You can adjust the waveforms of the three input variables by clicking "Details" for each. This will show a window that allows you to edit the intensity of each input variable over time. The sliders in the main window allow you to multiply your waveform by a certain factor.

For example, setting auxiliary power to 75 from 50 would scale all the values by 3/2. Look at the axes for changing numbers. When you're ready to test out your values, click "START" and watch the output by clicking "Output Graph." While the output graph is scrolling with data, the color of the plasma in the main window will change too in accordance to the score. You can repeat the process as much as you like. Please e-mail Daren Stotler with comments and feedback.
